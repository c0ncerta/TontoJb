# Changelog

All notable public-release changes are tracked here. This project uses research snapshots rather than stability promises.

## [0.1.0-research] - 2026-05-02

### Added

- Initial public research snapshot for the PS5 Netflix-sandbox TontoJB proof of concept.
- README overview, responsible-use notice, exploit-chain model, requirements, syscall table, limitations, and attribution.
- MIT License with additional research-context notice for public PoC publication.
- Public documentation map, research log, telemetry notes, run matrix, and post-twin notes.
- Repository hygiene rules for local certificates, runtime logs, `docs/otros/`, and imported reference material.

### Changed

- Moved auxiliary documentation and local-only notes out of the public docs surface.
- Added explicit attribution links for Netflix-N-Hack and Gezine / Y2JB.

### Scope

- Research target: PS5 firmware 11.60 under the documented Netflix sandbox conditions.
- Release status: proof of concept, not stable software and not a consumer jailbreak package.
